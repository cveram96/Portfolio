Predicción del valor mediano de viviendas en California

Este modelo analiza diversos factores socioeconómicos utilizando datos del censo de California. Se implementan técnicas de Machine Learning para la predicción del valor mediano de las viviendas.

  • Análisis de factores socioeconómicos.
  • Uso de datos del censo de California.
  • Implementación de técnicas de Machine Learning.
